Repository for Bot-related Research

This project was created to be a public repository to support bot-related research. You are welcome to share your research (including the data) with the bots researchers community. To add a new paper, you can always email us or simply add a paper using our online submission page. To cite this repository website please use the following BibTeX entry:

@misc{Bot_Research_Repository,
author = {DAS Lab},
title = {The Bot Repository: A repository of research articles on Software Bots},
publisher = {Data-driven Analysis of Software (DAS) Lab, Concordia University},
note = {http://das.encs.concordia.ca/}
}
